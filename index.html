Praticando com o Git e Github

O que é o GIT?

Git é uma plataforma que serve para criar versões de determinado software através de dos codigos de cada linguagem. Serve também para alterar codigos de versões de softwares ja criados.

O que é o Github

Github é um tipo de rede social para programadores, lá nós podemos olhar os codigos de outras pessoas, copiar ou até mesmo criar codigos e enviar através do Github desktop.
